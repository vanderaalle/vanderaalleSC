SO a class for the annotation of sound objects


Inherits from: Object : Model


The SO class is intended to store and manipulate sound object-based annotations  about a soundfile. 

It allows to describe the passed soundfile in terms of its component sound objects. 

Each sound object is described through the following parameters:

time properties

attack: attack os the sound in seconds

duration: duration of the sound in seconds


typo-morphological properties

profile: overall intensity morphology (range: -2.5, 2.5) 

caliber: caliber of its mass (range: 0.0, 2.0)

variation: changing rate of the mass (range: 0.0, 3.0)

site: starting register of the mass (from low) (range: 0.0, 7.0)

dynamics: perceived volume (range: 0.0, 7.0)


other

group: index (integer). Allows grouping of sound objects having the same group (i.e. "voices", "layers") 

text: remarks (string)


See also: SOSpace, SOScore  



Creation / Class Methods


*new (soundPath)

Creates a new SO and associates a sound file to be annotated.

soundPath - the path of the sound file. Default value is nil. 

If the path is not nil, the local server is booted and the file is loaded into a buffer.

It sets the soundPath var

// creating a SO object

o = SO.new("sounds/a11wlk01.wav") ;

// now:

o.soundPath 

// returns:

sounds/a11wlk01.wav


Accessing Instance and Class Variables


classDict

A dictionary containing the definition of typological classes in terms of ranges.

It is a classvar.

SO.classDict.keys.do({ arg soClass ;

soClass.post ; 

"\t".post ; 

SO.classDict[soClass].postln ;

})

buffer

The buffer where the sound file to be annotated is loaded.

You typically would not need to control it.

Default value is nil.

server

The instance of Server in use.

You typically would not need to control it.

Default value is nil. 

duration

The duration in seconds of the loaded sound file. 

It is setup when you load the file.

Default value is nil.

objectDict_(arg1)

objectDict

A dictionary storing all the sound objects in  the form: name -> objectDef, 

where objectDef is an array containing the sequence of parameter values.

Default value is nil.

soundPath

The path of the loaded sound file.

Default value is nil.



// getting some infos

o = SO.new("sounds/a11wlk01.wav") ;

o.buffer ;

o.server ;

o.duration ;

o.soundPath ;

Annotation and manipulation


The following methods allows adding/removing/editing sound objects and their properties.


loadSound (path)

Loads a sound file. It is called if you pass a path to the constructor.

But it can be called after too.

path - the path of the sound file to be loaded. Default value is nil.

// you can load a file after creation with loadSound  

o = SO.new ; // no file associated

o.loadSound("sounds/a11wlk01.wav") // loading the file and setting soundPath




play (selector, to)

Plays the file.

selector - selector allows to choose to play  sound objects or regions. Default value is 0. In this case, selector if interpreted as the attack time in seconds. If  selector class is Symbol or String, then selector is intepreted as a sound object name , which is then played.

to - the end of the region to be played in seconds. Default value is nil, in which case it defaults to sound duration.  In case selector is a symbol or a string, it is discarded.

// easier to show than to describe

o = SO.new("sounds/a11wlk01.wav") ;

o.add("[Ava]", 3, 0.85) ; // adding an object

o.play ; // play all (from = 0, to = sound duration) 

o.play(2, 3) // play region from 2 to 3 sec

o.play(2) // play region from 2 to ens

o.play("[Ava]") // play sound object \test



add (name, attack, duration, profile, caliber, variation, site, dynamics, group, text)

Adds a sound object to the objectDict.

name - string/symbol representing the name. Default value is nil. Strings are converted internally into symbols.

attack -  attack of the object in secs. Default value is nil.

duration - duration of the object in secs. Default value is nil.

profile - float in range(-2.5, 2.5). Default value is nil.

caliber - float in range(0.0, 2.0). Default value is nil. 

variation - float in range(0.0, 3.0). Default value is nil.

site - float in range(0.0, 7.0). Default value is nil. 

dynamics - float in range(0.0, 7.0). Default value is nil. 

group - integer in range(0.0, 7.0). . Default value is nil. Allows grouping (i.e. "voices", "layers")

text - string for remarks. Default value is nil. 

// inline example

g = SO.add;

g.doSomething; // explanation

g.cleanUpMethod;



remove (name)

Remove a sound object. 

name - the name of the sound object to be removed. Default value is nil. Can be string or symbol (internal format is symbol). If nil, all the objects are removes (clean method is called)

// inline example

o = SO.new("sounds/a11wlk01.wav") ;

o.add("[Ava]", 3, 0.85) ; // adding an object

o.add(\bip1, 0, 0.35) ;

o.add(\bip2, 0, 0.35) ;

o.print ;

o.remove("[Ava]") ; // adding an object

o.print ;

o.remove ;

o.print ;
